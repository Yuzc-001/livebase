# Source Preservation

Use this file when the main question is:

# how do we keep abstraction from drifting away from source truth?

## Core rule

Derived notes, summaries, tags, and AI outputs should not replace the original source.
They should remain attached to it.

## Preserve
- source document or note
- origin context
- relevant time/version context
- explicit link from abstraction back to source

## Failure signs

- summary exists but source is unclear
- AI output becomes the new truth layer
- later users cannot tell what was original versus derived
