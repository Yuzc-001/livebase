# Livebase v0.1 Rebuild Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild Livebase around resumable work so the local product centers on checkpoint, resume, contract, and writeback instead of generic memory storage.

**Architecture:** Replace the legacy `scripts/` and `schemas/` surface with a new `src/core` and `src/cli` structure. Keep the flat JSON store and Zod validation, but make the main loop task-centric by introducing `ExecutionContract` and `VerificationLedger` as first-class continuation protocols.

**Tech Stack:** Bun, TypeScript, Zod, flat JSON store
